UseTenantResolver

Adds a tenant resolver middleware that looks for the current tenant from the Uri.

It handle the requet with a StatusCode = 400 if not tenant is found unless that a tenant not found notification handles the request.

The default tenant provider does not filter un-provisioned tenants si it is up to the application to verify this if required.
