GetServerVersion

Gets the server version.

Signature:
public String GetServerVersion()
Returns

The server version.

Remarks

This method returns the compatibility level of the database configured in the DatabaseApi property as:
When 65 then SQL Server 6.5
When 70 then SQL Server 7.0
When 80 then SQL Server 2000
When 90 then SQL Server 2005
When 100 then SQL Server 2008/2008R2
When 110 then SQL Server 2012
When 120 then SQL Server 2014
When 130 then SQL Server 2016
When 140 then SQL Server 2017
When 150 thrn SQL Server 2019
Else new unknown.

Exceptions
Type Summary
SchemaManipulationException Thrown when there is an error accesing schema data.
