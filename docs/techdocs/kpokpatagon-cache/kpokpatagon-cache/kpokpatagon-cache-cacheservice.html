Example

Service configuration:

Cache must be configured within a "Caching" section in appsettings.json.


             "KpokPatagon": {
               "Caching": {
                 "DefaultSlidingExpireTime": "00.01:00:00",
                 "Redis": {
                   "Enabled": false,
                   "DatabaseId": -1,
                   "ConnectionString": "localhost"
                 }
               }
             }

Service registration:

This cache service must be registered with DI within the "configure service" method:


             services.AddKpokPatagon(<configuration>)
                     .AddCacheService();

Service usage:

Once registered just inject the service where required:


             public MyClass(ICacheService cacheService) { ... }
