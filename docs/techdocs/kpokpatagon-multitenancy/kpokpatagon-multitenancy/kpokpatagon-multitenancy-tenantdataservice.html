TenantDataService

TenantDataService implements the data services tenants table.

Namespace:
KpokPatagon.Multitenancy
Type:
Class
Public:
Yes
Protected:
No
Sealed:
No
Abstract:
No

Remarks

This service does not have any configuration for the services required for it to work.

All required services like the CommandBuilder and DataModelCommandBuilder must be configured by a DataServiceOptions<TenantDataService> configuration within the dependency injection service collection.
